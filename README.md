# my_algorithm_lib

自分専用のアルゴリズムライブラリ。学習用。
再帰アルゴリズムの不変構造を非再帰で統一的に扱う試み。

## 目的

- 再帰アルゴリズムの普遍構造を理解する
- 非再帰化の技法を体系化する
- DFS / 木DP / グラフDP / AST などを統一的に扱う
- 自分のアルゴリズム理解を深める

## 概要

再帰アルゴリズムを次の形として捉える。
```
enter(node);     // 行きがけ処理
for child in children(node):
    recurse(child);
exit(node);      // 帰りがけ処理
```
